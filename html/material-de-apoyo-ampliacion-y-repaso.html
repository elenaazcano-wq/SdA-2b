Material de apoyo, ampliación y repaso

Antes de empezar

Para recordar antes de empezar
¿Qué significa comunicarse bien?
Comunicarse bien significa expresar ideas, sentimientos o información de forma clara para que otras personas puedan entendernos. También significa escuchar con atención, respetar los turnos de palabra y adaptar el mensaje a la situación.

Claves para una buena comunicación

  • Hablar con claridad.
  • Organizar las ideas antes de expresarlas.
  • Escuchar con atención.
  • Utilizar un tono de voz adecuado.
  • Revisar lo que decimos o escribimos.
  • Acompañar el mensaje con apoyos visuales si es necesario.

Apoyos

Apoyo 1. Cómo encontrar la idea principal
La idea principal es la información más importante de un texto o una explicación. Para encontrarla, puedes hacerte estas preguntas:

  • ¿De qué trata este texto?
  • ¿Qué es lo más importante que quiere explicar?
  • Si solo pudiera contar una idea, ¿cuál sería?

Estrategia útil

  1. Lee el texto completo.
  2. Subraya las palabras más importantes.
  3. Busca la frase que mejor resume el contenido.
  4. Comprueba si las demás ideas dependen de esa.

Apoyo 2. Cómo organizar la información

Antes de hacer una infografía o grabar un podcast, conviene ordenar las ideas.

Pasos para organizar la información

  1. Escribe el tema principal.
  2. Anota tres ideas importantes.
  3. Añade un ejemplo para cada idea.
  4. Busca una imagen relacionada con cada parte.
  5. Decide el orden en el que lo vas a explicar.

Apoyo 3. Cómo hablar de forma clara
Para hablar bien en una exposición o en un podcast es importante cuidar la voz y el mensaje.

Consejos para hablar mejor
Respira antes de empezar.
Habla despacio.
Pronuncia bien las palabras.
Mira al público si haces una exposición.
No leas demasiado rápido.
Usa frases cortas y claras.
Haz una pequeña pausa entre ideas.
 
Apoyo 4. Cómo revisar un trabajo
Revisar sirve para mejorar. No es repetir, sino comprobar si el trabajo está bien hecho.

Lista de revisión para la infografía
Tiene un título claro.
Explica tres ideas importantes.
Incluye imágenes relacionadas.
Se entiende con facilidad.
Está ordenada.
Tiene un elemento interactivo.
Lista de revisión para el podcast
Se escucha bien la voz.
La duración es adecuada.
El mensaje está ordenado.
Incluye presentación, desarrollo y cierre.
El grupo ha ensayado antes de grabar.
 
Apoyo 5. Frases útiles para trabajar en grupo
Para proponer ideas
Yo pienso que…
Podemos hacerlo así…
Mi propuesta es…

Para pedir ayuda
¿Me lo puedes explicar otra vez?

¿Podemos hacerlo juntos?

Para llegar a acuerdos
Estoy de acuerdo porque…
Podemos elegir esta opción.
Vamos a decidir entre estas ideas.
Esta propuesta puede funcionar bien.
Para hacer una coevaluación respetuosa
Me ha gustado…
Está muy bien explicado…
Podría mejorar si…
Tal vez podéis cambiar…
